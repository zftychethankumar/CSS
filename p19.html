CSS PSEUDO-CLASSES..!

he CSS pseudo-classes allow you to style the elements or part of the element without adding any IDs or classes to them.

CSS3 introduced a new double-colon (::) syntax for pseudo-elements to

selector::pseudo-class {property: value;}
